# Financy Forecast (TanStack Start)

This app is the new TanStack Start target for the Financy migration.

## Scripts

- `bun run dev` starts the app on port `3056`
- `bun run build` creates a production build
- `bun run start:prod` starts the Nitro server on port `4056`
- `bun run check-types` runs TypeScript checks
- `bun run lint` runs ESLint
